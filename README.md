# FE_Lectures

Live-coded front-end lessons I use to **teach students to code** — the actual code typed during each class, kept so students can revisit exactly what we built and how we got there.

## What's here

- **`Lectures/`** — Code typed live during each lesson, organized in dated folders so a student can follow the progression class by class.
- **`Resources/`** — Supporting docs, references, and tools for the course.

## How it's meant to be used

This isn't a polished library — it's a teaching record. Each folder captures a real session: the false starts, the refactors, and the working result. Students clone it to:

- Replay a lesson at their own pace
- Compare their version against what we wrote together
- Grab a starting point for the day's exercise

If you're learning from these, the goal is never to copy the final file — it's to **type it yourself** and understand each step. That's the same philosophy behind everything I build.
